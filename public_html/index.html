AI Chatbot

This is a project that allows users to chat with an AI Bot. Using OpenAI's API, I created a full-stack web application that allows users to securely create or login to an account, chat with AI, save conversations, delete conversations, etc. This application is completely secure using JWT tokens, HTTP-only cookies, password encryption, and middleware chains. For this application, I used the MERN stack, using MongoDB for the backend and Node.js for runtime environment. I used Express.js to handle routing and middlewares, and the front-end was handled with React.js. Check it out!
